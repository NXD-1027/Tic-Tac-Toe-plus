# v0.1 项目骨架与棋盘渲染实现文档

## 1. 版本目标

v0.1 的目标是搭建项目基础结构，并完成一个可以在浏览器中打开的静态页面。页面需要显示完整的超级井字棋棋盘，也就是 9 个 3×3 小棋盘组成的 9×9 棋盘。

这个版本暂时不要求完整游戏规则，只要求完成页面、文件结构和基础棋盘渲染。

---

## 2. 本阶段要完成的内容

- 创建基础项目文件。
- 创建页面结构。
- 创建基础样式。
- 使用 JavaScript 动态生成 9×9 棋盘。
- 每个格子带有必要的数据属性，方便后续点击落子。
- 页面包含标题、状态栏、棋盘区域和控制按钮。

## 4. 页面结构设计

`index.html` 建议包含以下区域：

```text
header：游戏标题和简介
status：当前玩家、下一步提示
main：棋盘容器
controls：重新开始、悔棋、模式选择按钮
```

推荐结构：

```html
<div class="app">
  <header class="app-header">
    <h1>超级井字棋</h1>
    <p>Ultimate Tic-Tac-Toe</p>
  </header>

  <section class="status-panel">
    <div id="current-player">当前玩家：X</div>
    <div id="next-board-hint">当前可在任意小棋盘落子</div>
  </section>

  <main>
    <div id="board" class="macro-board"></div>
  </main>

  <section class="control-panel">
    <button id="restart-button">重新开始</button>
    <button id="undo-button">悔棋</button>
    <select id="mode-select">
      <option value="local">本地双人</option>
      <option value="ai">人机对战</option>
    </select>
  </section>
</div>
```

## 5. 棋盘 DOM 设计

整个棋盘分为两层：

1. `.macro-board`：3×3 大棋盘容器。
2. `.micro-board`：每个大格子内部的 3×3 小棋盘。
3. `.cell`：小棋盘内的单个格子。

每个 cell 建议携带以下数据属性：

```html
<button
  class="cell"
  data-macro-row="0"
  data-macro-col="0"
  data-micro-index="0"
></button>
```

后续点击时可以通过这些数据快速确定用户点击了哪个位置。

## 6. JavaScript 模块职责

### 6.1 main.js

负责项目初始化：

- 页面加载后调用初始化函数。
- 调用棋盘渲染函数。
- 绑定按钮事件。

### 6.2 render.js

负责渲染相关功能：

- 创建棋盘 DOM。
- 更新状态栏文字。
- 后续负责根据 gameState 刷新 UI。

### 6.3 game.js

本阶段可以只负责创建初始状态：

- `createInitialState()`
- `resetGame()`

### 6.4 utils.js

本阶段可以加入坐标转换函数：

- `microIndexToRowCol(index)`
- `rowColToMicroIndex(row, col)`

### 6.5 ai.js

本阶段预留文件，暂不实现 AI。

## 7. 样式要求

v0.1 的样式目标是让棋盘清晰可见。

建议：

- 页面居中显示。
- 大棋盘使用 CSS Grid，3 列 3 行。
- 小棋盘也使用 CSS Grid，3 列 3 行。
- 小棋盘之间用较粗边框区分。
- 小格子大小一致。
- 鼠标悬停时有轻微高亮。
- 移动端下棋盘宽度不超过屏幕。

推荐关键样式：

```css
.macro-board {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 8px;
}

.micro-board {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  border: 2px solid #333;
}

.cell {
  aspect-ratio: 1;
  border: 1px solid #aaa;
  font-size: 1.5rem;
}
```

- AI。
